# TSP MPI Evolution Experiment

This directory contains the serial and MPI DEA implementations for the parallel and distributed computing final project.

## Directory Layout

- `src/` - C source files, including the original `TSP0.C`, fixed serial versions, and MPI DEA version.
- `data/` - TSP input data such as `pcb442.tsp`.
- `docs/` - Reference paper and project notes.
- `scripts/` - PowerShell and Python automation scripts.
- `results/` - Generated CSV/TXT experiment and analysis outputs.
- `bin/` - Compiled executables generated by the scripts.

## Common Commands

Run the formal 10-seed serial vs DEA experiment:

```powershell
powershell -ExecutionPolicy Bypass -File .\scripts\run_experiments.ps1
```

Analyze the formal experiment results:

```powershell
python .\scripts\analyze_results.py
```

Run the quick end-to-end pipeline smoke test:

```powershell
powershell -ExecutionPolicy Bypass -File .\scripts\smoke_test_experiment_pipeline.ps1
```

Run individual smoke tests:

```powershell
powershell -ExecutionPolicy Bypass -File .\scripts\smoke_test_serial.ps1
powershell -ExecutionPolicy Bypass -File .\scripts\smoke_test_dea.ps1
```

## Notes

- Do not edit `src/TSP0.C`; it is kept as the original reference code.
- Generated experiment outputs should go under `results/`.
- Generated executables should go under `bin/`.
- The current implemented parallel algorithm is DEA only. HDEA and moving colony HDEA have not been implemented.
